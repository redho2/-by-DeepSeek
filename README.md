# 🏜️ 沙子俄罗斯方块 · Sand Tetris

一个融合**沙子物理**与**连通消除**机制的俄罗斯方块游戏。  
方块由彩色粒子组成，落地后像真实的沙子一样流动、堆积。  
只有**从左边缘连通到右边缘**的同色块才会消除！

---

## 🎮 操控方式

| 平台 | 操作说明 |
|------|----------|
| ⌨️ **键盘** | `←` `→` 移动 · `↑` 旋转 · `↓` 软降 · `空格` 硬降 · `R` 重启 |
| 🎮 **手柄** | 方向键移动 · `A` 旋转 · `B` 软降 · `X`/`LB` 硬降 · `Start` 重启 |
| 📱 **触屏** | 使用屏幕下方的按钮直接操控 |

---

## ✨ 游戏特色

- 🧱 **沙子粒子物理**：每个方块由若干微小粒子组成，落地后会自然滑落堆积。
- 🌈 **四种高对比度颜色**：红、银、绿、蓝，极易区分。
- 🔗 **连通消除**：只有从左边缘接触到右边缘的同色粒子块才会被消除，策略性更强。
- ⚡ **极速性能**：代码经过极限优化，流畅运行于移动端和桌面端。
- 🕹️ **全平台兼容**：同时支持键盘、手柄和触摸屏操作。
- 📱 **移动端适配**：响应式布局，自动适应各种屏幕尺寸。
